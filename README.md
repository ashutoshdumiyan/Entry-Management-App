
# Description
Innovaccer is an android application used to manage entries of visitors. On arrival, the visitor will enter their details along with their host's details. When the visitor checks in, the host will receive the details os visitor by E-Mail and SMS. When the visit is over, the visitor can check-out and their visit's details will be sent to them via an E-Mail.

Platform - Android <br/>
Frontend - Java <br/>
Backend - Firebase(Firestore) <br/>

# Installation

The app can be installed on any android device by downloading the apk installer file from the "Installer" folder.
You can also install it in your device/emulator by cloning this repository.

# App's usage tips
The first screen that will open up when the app launches is the form which the visitor will fill and after clicking the "Check In" button, the details will be stored in a databse and an E-Mail ans SMS will be sent to the host informing them of the visitor's arrival.
